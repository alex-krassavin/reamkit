# Documentation

## API reference (generated)

`docs/reference/` holds the Markdown API reference, generated from the public
TypeScript surface (`src/index.ts` + the `reamjs/document-model` subpath) and its
TSDoc comments via `@tanstack/config/typedoc` (TypeDoc + typedoc-plugin-markdown
+ frontmatter).

It is **not committed** (regenerable artifact). Generate it with:

```sh
npm run docs:api
```

Every page is autogenerated — "DO NOT EDIT" — and carries frontmatter (`id`,
`title`) ready to drop into a Markdown docs site (Docusaurus, Nextra, Astro
Starlight, VitePress, …). GitHub source links resolve once the `origin` remote
is set.
